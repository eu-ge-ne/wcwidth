## Installation

### Deno

```bash
deno add jsr:@eu-ge-ne/wcwidth
```

### Node.js

```bash
# pnpm
pnpm i jsr:@eu-ge-ne/wcwidth

# yarn
yarn add jsr:@eu-ge-ne/wcwidth

# npm
npx jsr add @eu-ge-ne/wcwidth
```

### Bun

```bash
bunx jsr add @eu-ge-ne/wcwidth
```
